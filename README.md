# HSLuv - Human-friendly HSL

This website is hosted on [https://www.hsluv.org](https://www.hsluv.org)

The reference implementation is [written in Haxe](https://github.com/hsluv/hsluv-haxe).

To build website:

```
npm run build
```

To start localhost server:

```
npm run serve
```
